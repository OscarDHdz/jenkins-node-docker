# Jenkins-docker node CI/CD example
Basic express API example used for Jenkins-Docker example with Job, DSL Job and Pipeline

## Getting Started
Install dependencies  
```
npm install
```

Run server:  
```
npm start
```

### Test Available
Run  
```
npm test
```
